# Audio Snipper

Audio Snipper is a lightweight WPF application for capturing short audio clips from the system input, saving them to disk, and performing simple trimming/snipping operations.

## Key Features
- Record audio clips from the default input device.
- Visual snipping interface to select and extract a portion of a recording.
- Save recordings to common audio formats.
- Simple, dependency-minimal WPF UI targeting .NET 10 and C# 14.0.

## Requirements
- Windows 10/11 or later
- .NET 10 runtime / SDK

## Building from Source
1. Clone the repository:

2. Open the solution in Visual Studio: use __File > Open > Project/Solution__ and select the `.sln` file.

3. Restore NuGet packages if required: use the __NuGet Package Manager__ or __Project > Restore NuGet Packages__.

4. Build the solution: use __Build > Build Solution__.
